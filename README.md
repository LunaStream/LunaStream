# LunaStream
A lavalink alternative focus on stability, decent speed and modulize like FrequenC

# Development

## How to setup
```
git clone https://github.com/LunaticSea/LunaStream
cd LunaticSea
lit install
```

## How to run development enviroment
```
luvit dev
```

## How to build binary
```
luvit make
```

## How to build manifest
```
luvit make manifest
```
